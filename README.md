# assetfix

Enables loading loose game asset files (models, textures, etc.) from the `Data/` directory in the WoW 1.12.1 client, and allows multi-character `patch` archive names (e.g. `patch-12.mpq`, `patch-jimbo.mpq`).

This project has moved to [WeirdUtils](https://codeberg.org/MarcelineVQ/WeirdUtils/).
